# Shizuka Images Setup

## Required Image: hearts-background.jpg

Please save the beautiful hearts background image that you provided as:
`dashboard/images/hearts-background.jpg`

This image shows:
- Beautiful pink/purple gradient background
- Floating hearts of various sizes and colors
- Sparkles and light effects
- Perfect romantic/magical atmosphere
- Ideal for the main dashboard background

## Current Setup:
- The dashboard is configured to use this image as the main background
- Beautiful gradient overlay preserves readability
- Existing logo images are used for floating elements and hero character

## How to add:
1. Save the beautiful hearts background image as `hearts-background.jpg` in this folder
2. The dashboard will automatically use it as the background
3. No code changes needed - just add the image file
4. The image will appear with a subtle overlay to maintain text readability

## Logo Image: new-logo.jpg

To replace the current logo with a new image:
1. Save your new logo image as `new-logo.jpg` in this folder
2. The dashboard will automatically use it for:
   - Navigation logo (top left)
   - Hero section character (large image on right)
   - Floating background elements
3. Supports JPG, PNG, or other web formats
4. Recommended size: 500x500px or similar square format

## Fallback:
If the image isn't found, the dashboard will show the gradient background instead.
